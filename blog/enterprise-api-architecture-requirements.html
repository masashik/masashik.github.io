Enterprise API Architecture Requirements

Last updated: 6th May 2021 Filed under: career

Requirements

  • API Framework development as cloud-native using dropwizard/spring-boot/micronaut and trained fault-tolerant for SLA
  • scalable (MQ to connect to DB)
  • SQL connection pool
  • thread pool for each API instance
  • fault-tolerant
  • Metrics
  • language agnostic (Java/JS/Golang)
  • compatible for OAuth2.0
  • compatible for API-to-API call
  • enabled for Swagger UI
  • Netflix GraphQL with Auth token propagating techniques
